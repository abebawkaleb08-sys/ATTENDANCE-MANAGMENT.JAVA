📚 Attendance Management System (Java OOP Project)
📌 Project Overview
This project is a simple Java Object-Oriented Programming (OOP) application that simulates an attendance management system for students and teachers. It demonstrates core OOP principles such as inheritance, polymorphism, encapsulation, method overloading, and ArrayList usage.

🧠 OOP Concepts Used
Inheritance → Student and Teacher classes extend Person class
Encapsulation → Use of private and protected variables
Method Overriding → displayInfo() is customized in subclasses.


Method Overloading → markAttendance() methods in AttendanceManager
Collections → ArrayList used to store students
User Input → Scanner class for runtime input

🏗 Project Structure
Person → Base class containing common attributes (name, id)
Student → Extends Person, adds attendance tracking
Teacher → Extends Person, custom display behavior
AttendanceManager → Handles student list and attendance logic
Main → Entry point, handles user interaction

⚙️ Features
Add students to the system
Track student attendance
Mark attendance for multiple days
Display student and teacher information
Interactive user input via console.
